sty

OS/161 Reference Manual

Name

sty - run some hogs

Synopsis

/testbin/sty

Description

sty runs six copies of hog.

Requirements

sty uses the following system calls:
  • fork
  • execv
  • write
  • _exit
sty is only likely to be useful for testing the scheduler.

Bugs

sty does not wait for its child processes to finish.
