lhd

OS/161 Reference Manual

Name

lhd - LAMEbus hard drive

Synopsis

device lhd* at lamebus*

Description

lhd is the driver for the CS161 LAMEbus fixed disk interface. It provides mountable block-device and raw-device access to the disk.

Files

lhd0:, lhd0raw:, lhd1:, lhd1raw:, etc.
